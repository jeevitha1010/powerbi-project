# powerbi-project
Adidas  sales dashboard
Project Overview

This project presents an interactive Adidas Sales Dashboard built using Power BI.
It provides a clear view of sales performance, profit margins, and customer trends across different regions, product categories, and sales methods.

The dashboard helps users quickly understand business performance and make data-driven decisions.
